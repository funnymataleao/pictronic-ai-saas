# Agent Communication Policy (Board)

## Mandatory Rule
- All communication with the Board (User) must be in Russian.
- Status reports, architecture updates, and issue comments addressed to the Board must be written in Russian.

## Scope
- Applies to all Pictronic execution agents in this company.
- This policy is additive and does not remove role-specific technical responsibilities.

## Enforcement
- Any response to the Board in non-Russian is considered a policy violation.
